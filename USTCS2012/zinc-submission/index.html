ZINC Automatic Grading System



Introduction

ZINC stands for "ZINC is not CASS". It is an online assessment platform that allows you to submit your programming work for automated grading. Upon completion of grading, you will get back a report showing the results of the automated assessments.

End of Introduction

Prepare a Zip File with your Source Code


As ZINC only accepts zip file, you will need to prepare a zip file with your source code as described below. Note that you must compress the required files only, NOT a folder containing the files. Also, do not include any file that is not required. Check the lab/assignment page for the exact list of required files.

The following is for the Windows user, although the steps are very similar on MacOS as well. MacOS users may refer to this short youtube video on compressing multiple files on MacOS. Note for MacOS users: MacOS may add some "extra" stuff like "__MACOSX" to your zip file. Zinc may report that those are "extra files". You may just ignore those "errors" and check the test case results. If you really need to remove those extra files because it affects the test cases somehow, you may follow this guide.

  1. Go to the folder where your source code is located.

  2. Suppose you want to submit your work for Lab 1 and let's say you only need to submit one .cpp file. Right-click the source file that you need to submit, click "Send to" and "Compressed (zipped) folder".

    If you need to submit multiple files, select/highlight all those files first (hold down ctrl key on Windows, and click on them one by one; or, just hold down your mouse to select a group of files), before you right-click one of them for compressing all selected files.



    Now, you should see a zip file with your source code.

End of Zip File Preparation

How to Submit Your Work


  1. Go to https://zinc.cse.ust.hk

  2. Login to ZINC using your ITSC credentials.


  3. If you have successfully login, you should see the following main page.


  4. Click COMP 2012 and this brings you to the assignment submission page. Note: the following screenshots show COMP2012H as an example just for illustration purposes, but all steps apply to COMP2012 as well.


  5. Click on the lab/assignment that you want to submit your work to. Suppose you want to submit Lab 1, click "Lab 1: Makefile and Separate Compilation". Now, you should a page like this. Don't mind the description there (if any), they are just copied and pasted from the lab/assignment page. Our lab/assignment page is always the most up-to-date one.


  6. To submit your work, click the blue button on the top-right hand corner of the page and a file selection dialog is opened.


  7. Go to the folder where you put your zip file, select the zip file (the one with your source code), and click "Open".



    You will be prompted and asked whether you want to upload the selected file. To confirm, click "Upload".

  8. After the zip file has been uploaded, you will see the following page.


  9. ZINC should start grading your submitted lab/assignment. Please note that it normally takes a few seconds to complete the grading. Press "F5" to refresh the page and see whether the grading is done. When the grading is done, you should see the following.



    Note: Your score is shown under "Reports - Score".

  10. If you would like to check the grade details, click on the score and you will see the report.


  11. Expand the menu item for more details. E.g., more details about compilation.

End of How to Submit Your Work
