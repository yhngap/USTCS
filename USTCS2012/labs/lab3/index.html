Lab 3 More Class Construction and Destruction

Pointset

Objective

In this lab, you are going to practice using class constructors and destructors by implementing a class called Pointset. You will also practice on a technique that is useful for the first assignment - to maintain a dynamic array just big enough to contain all the items (e.g. points, for this lab).

Description

A PointSet is a set of 3D points, it is backed by a dynamic array of 3D point objects and each 3D point object has three double member variables: x, y, and z, which represents the x-coordinate, y-coordinate and z-coordinate of a 3D point.

Download the starting code, and complete the TODOs in PointSet.cpp. You should read all the given source code especially the PointSet.h as well as the sample output to understand the requirements.

Note: if you use VS Code, please put all the starting code files to its own project folder, then open that folder in VS Code. To compile, open a terminal and use the command g++ -std=c++11 *.cpp -o programName.exe to compile all cpp source files in the current folder into a program named programName. You may use the command ./programName.exe in terminal to run the program then. Alternatively, you can also create a Makefile and make use of it. Tips: press up arrow (multiple times for earlier commands) to repeat the last command(s) in terminal.
