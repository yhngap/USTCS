Lab 10 Binary Trees and Binary Search Trees

Binary Trees

Objective

In this lab, you will implement several functions of Binary Tree and a class of Binary Search Tree.

Description

A binary tree is a tree data structure in which each node has at most two children nodes, which are referred to as the left child and the right child. A basic binary tree node implementation is given in (BtreeNode.h)

DO NOT modify this file. There are three helper functions provided, which are useful in completing this lab.

  • bool BtreeNode::is_ancestor_of(const T& data) returns true if the current node is an ancestor of any node that has data.
  • const T BtreeNode::find_max() and const T BtreeNode::find_min() recursively search the children to find the min and max element.

Binary Tree Operations

  • tree_height : Calculate the height of a binary tree. The height of a tree is the number of edges on the longest downward path between the root and a leaf.
  • count_nodes : Calculate the number of nodes of a binary tree.
  • mirror : Create a "new" mirror tree. For example:

Binary Search Tree

A binary search tree is a rooted binary tree, whose key value in each node must be greater than or equal to any key value stored in the left sub-tree, and less than or equal to any key value stored in the right sub-tree.

In this lab, we index the binary tree by two keys(values); each BinaryNode object in BinarySearchTree class contains two comparable data members T1: x, T2: y, which satisfy the following:

  • The x value of any of the left sub-tree node is less than or equal to its parent node's x value. If they are the same, then the y value of the left sub-tree node is less than or equal to its parent's y value.
  • The x value of any of the right sub-tree node is greater than or equal to its parent node's x value. If they are the same, then the y value of the right sub-tree node is grater than its parent's y value.

The BinarySearchTree has the following member functions

  • insert(const T1 &x, const T2 &y): Insert a pair of key to the BST. If the entry already exists, you don't need to insert it again, i.e., there are no duplicate nodes.
  • contains(const T1 &x, const T2 &y): Check if the BST contains the value (x,y).
  • print_max(): print out the maximum value pair stored in the BST. If two nodes have the same maximum x value, find the maximum y among them.
  • make_empty(): Delete all nodes in the BST object.
  • print_tree(): Print the BST, the output is rotated -90 degrees. This function is provided to you.
