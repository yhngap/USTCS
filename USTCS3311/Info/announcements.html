August 31

DOWNLOAD AND INSTALL REQUIRED SOFTWARE

Before the first lab, download and install on your computer Oracle SQL Developer which can be downloaded from https://www.oracle.com/tools/downloads/sqldev-downloads.html. It is free, but you are required to create an Oracle account to download the software. There are versions for Windows, Mac and Linux. The latest version is 20.2 and JDK 8 or 11 is required to also be installed. If you have not already done so, you should also install the HKUST VPN software from http://itsc.ust.hk/apps/vpn/ as it will be needed in the labs to access the department's Oracle Database server.
August 31

LECTURE / TUTORIAL / LAB EXERCISE REQUIREMENTS

Almost all lectures, all tutorials and all labs will have exercises for you to complete and upload to Canvas by 11 p.m. on the same day. If you print the exercise worksheets or do the exercises on paper, then you will need to have access to a scanner or scanner app to scan the completed exercise for upload to Canvas. Only PDF files will be accepted. Your scanned exercise must be legible. If it is not legible, it will not be counted as submitted. Therefore, it is your responsibility to check the legibility of your scanned exercise before uploading it to Canvas. At most 2 upload attempts will be allowed for each exercise and no late submissions or submissions by email will be accepted.

If you are using a phone or tablet to scan the exercises, we highly recommend that you use the free scanning app Microsoft Office Lens, which is available for both Android and iOS devices.
