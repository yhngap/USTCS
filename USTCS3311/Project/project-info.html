OVERVIEW

The course project involves designing, implementing and querying a database using the Oracle database management system. This is an individual project, which means that your design and implementation should be your own work .

You must produce your design and implementation according to the schedule of tasks described below. The schedule includes several milestones, each of which is represented by a specific artifact. Additional information about what we expect from each artifact will be provided as required.

DESCRIPTION

The course project description describes the application and outlines the high-level requirements of the course project.

TASK GRADING

For Task 1, grading is based on both content and presentation. The content grade includes completeness and correctness of the design. The presentation grade includes readability and layout of the design. For Tasks 2 and 3 grading is based on the correctness of the SQL statements.
