COMP4021 Lab 2 jQuery Autocomplete

Overview

  • In this lab, you will learn how to create a custom autocomplete widget by jQuery. This lab is a starting point of project 1.
  • Several files (task_1.html, task_2.html and some libraries and style sheets) are given to you as a starting point here.

1. Demonstration of default autocomplete widget

  • Open default_autocomplete_example.html in any text editor.
  • You can see an array which is created through javascript by the following code. 
    var input_data = [
      "actionscript",
      "applescript",
      "asp",
      "basic",
      "c",
      "c++",
	  ...
      "scheme"
    ];
  • Right below the array input_data, you can see the following code, 
    $( "#search" ).autocomplete({
	source: input_data
});
    • $( "#search" ) is a jquery selector which is to select the object with id search.
    • $( "#search" ).autocomplete(...) is going to create the autocomplete widget for the object with id search.
    • Inside the autocomplete( ... ), there is source: input_data, which is to indicate that input_data is the source data which will be filtered by autocomplete widget and then shows the suggestions.
  • Open task_1.html in a browser to see the effect of autocomplete widget.
