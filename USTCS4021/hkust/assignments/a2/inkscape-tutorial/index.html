Obtain SVG code from Inkscape

This tutorial shows you how to get SVG code from Inkscape, so that it can be used in your game development.

Steps

  1. Open Inkscape. Before drawing the picture, go to File->Document Properties, set Display units to px, set Scale x to 1.
  2. Draw the picture. You can scale it to the size you want afterwards.
  3. Go to File->Document Properties. Resize the page to fit content.
  4. Save the picture to an *.svg file.
  5. Open the *.svg file with an editor. Copy the attributes of the <svg> tag that starts with "xmlns" to your own <svg> tag. Copy the code between </metadata> and </svg> and enclose it with <g> ... </g> for drawing the object.
