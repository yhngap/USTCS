Description


  1. Note: unlike C++, each variable and method of a Java program needs to be defined in a Java class. No variable nor method could be defined independently out of all the classes in Java.

  2. The given C++ program is a quick sorting program with 4 functions (swap(), partition(), quickSort() and printArray()) and the main() function.

  3. You are required to read the given C++ code and convert it into the Java code that works exactly in the same way.


  4. As mentioned earlier, every variable and method in Java is declared inside a class, so we first start with creating a class:

     public class Main { ... }

  5. For the global functions in the C++ program, we declare it as a static function of Main class, where static is the keyword explained here explanation.

  6. Then we need to define the methods swap(), partition(), quickSort() and printArray and main() according to the C++ program. All these methods are needed to start with public static void in their headers (i.e. they should be class methods), otherwise you will need to create an object of the class Lab2 before you can call the methods.



    Note: You are required to keep the method name in camel-case.

 Start


  1. We have already created Main class together with quickSort(), printArray() and main() method for you.

  2. You are required to complete the Main class so that a quick sort algorithm is implemented.

  3. After your implementation, you should run the provided tests in QuickSortTests class. Be sure to pass all the tests before submission.

  4. It is worthwhile to mention that in C++, the main() function always starts with int main(), in Java, it always starts like the below:

    public static void main (String[] args)

    As you may notice yourself, no value could be returned by the main() method in Java.

  5. In C++ we can use sizeof(A)/sizeof(*A) in the main() function to determine the number of elements in A[] (must not do this after you have passed A as a parameter to a function, because A will then be reduced to a pointer, and sizeof() will not be able to get the number of bytes occupied by the whole array), in Java we can simply use the length attribute on any array A[].

  6. In C++ the statement int A[10];, will create an int array A[] for holding 10 integers. In Java, an array is an object (inherits from the java.lang.Object class) we need to use the new keyword to create an array as shown below:

    int [] A=new int[10];
